# Lunar Calendar Newtab - Svelte Version

Чистая переписка интерфейса на Svelte.

## Структура

```
newtab-svelte/
├── src/
│   ├── App.svelte          # Главный компонент
│   ├── Moon.svelte          # Пульсирующая луна
│   ├── DayInfo.svelte       # Информация о дне
│   ├── Palette.svelte       # Панель с цветами
│   └── main.js              # Точка входа
├── index.html               # HTML страница
├── package.json             # Зависимости
└── vite.config.js           # Конфиг сборки
```

## Разработка

```bash
# Установка зависимостей
npm install

# Dev-сервер с hot reload
npm run dev

# Сборка в production
npm run build

# Сборка с автообновлением
npm run watch
```

## Собранные файлы

После `npm run build` создаётся папка `../newtab-dist/`:
- `index.html` - главная страница
- `bundle.js` - скомпилированный JavaScript (~47KB)
- `bundle.css` - стили (~3KB)

## Компоненты

### Moon.svelte
- Большая пульсирующая луна в центре
- Анимация через CSS keyframes
- Цвета из палитры через props

### DayInfo.svelte
- Номер лунного дня (большой)
- Планета дня
- Фаза луны
- Процент освещённости

### Palette.svelte
- 12 кружков с цветами градиента
- Клик копирует цвет в буфер обмена
- Hover эффект

### App.svelte
- Управление состоянием
- Загрузка данных из background script
- Обновление фона по палитре
